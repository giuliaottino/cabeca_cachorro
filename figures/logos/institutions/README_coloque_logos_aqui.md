# Logos pendentes das instituições

Coloque aqui as logos das instituições participantes e parceiras.

O `index.qmd` usa os caminhos definidos em `_input_data/institutions.yml`. Quando uma logo não existe, o card usa automaticamente uma caixa com a sigla da instituição.

Sugestão: usar PNG ou SVG com fundo transparente, preferencialmente versão horizontal ou institucional simplificada.
